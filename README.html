Running a Full Demo

  1. Install (one time):
    bash cd .. # from the Promo/ folder, go to project root python setup.py

  2. Activate virtual environment:
    bash source ../venv/bin/activate

  3. Run against sample Timeline database:
    bash python ../vatican.py --path "Promo/Demo_Data/Test System/ActivitiesCache.db"

  4. Run interactive menu:
    bash python ../vatican_vault.py

  5. Run the demo script:
    bash python Promo/Demo_Scripts/demo.py
