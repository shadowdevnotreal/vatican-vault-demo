1.Executive Summary

This Operational Technology (OT) / Industrial Control Systems (ICS) security assessment was conducted pursuant to NERC CIP-002 through CIP-014 reliability standards and NIST SP 800-82 Rev. 3 (Guide to OT Security). 2.Purdue Model Architecture & Network Segmentation

The Purdue Enterprise Reference Architecture (PERA) defines security zones for industrial environments. Vatican Vault's analysis was scoped to Levels 3–4 (IT/OT demilitarized zone and site operations) where Windows endpoint activity data is available. OT protocol traffic (Modbus, DNP3, IEC 61850) is outside Vatican Vault's current scope but is covered by dedicated OT network monitoring.
